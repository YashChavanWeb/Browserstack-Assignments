## Test Management

### Core Features
* **Migrations:** Import from CSV, TestRail, Xray, and Zephyr.
* **Authoring:** Rich text editor, shared steps, and reusable templates.
* **Management:** Bulk edit, filters, and full version control.
* **Test Plans & Runs:**
    * Import JUnit XML/BDD JSON (50+ frameworks) or via cURL API.
    * Live execution with environment-specific configurations.
    * SDK integration for direct observability and auto-sync.
    * Dynamic test case selection for targeted execution.
* **Automation:**
    * Test ID tagging for history and traceability.
    * Automated flaky test and failure pattern detection.
    * Failure grouping by common root cause.
    * Custom alerts (Slack, Email, etc.).

---

### AI Agents
* **Generator:** Creates tests from CSV, prompts, Figma, PRDs, or images.
* **De-duplication:** Merges or removes semantically similar tests.
* **Low-Code:** Converts manual steps into low-code automation.
* **Failure Analysis:** AI-driven diagnostics for failed runs.
* **Test Selection:** Contextual selection based on Jira tickets.
* **Data Gen:** Auto-generates synthetic data for test cases.
* **Maintenance:** AI-assisted planning and test script updates.
